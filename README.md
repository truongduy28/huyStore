### Prerequisites

NPM / Yarn and Node.js installed

### Installing (only first time)

cd client && npm install

```
cd server && npm install
```

### Running the app

Open a terminal on server directory

```
npm run start:dev
```

and open another terminal on client directory

```
npm run start
```
